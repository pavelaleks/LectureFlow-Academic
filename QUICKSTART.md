# Quick Start Guide

## Первые шаги

1. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Настройте переменные окружения:**
   - Создайте файл `.env` в корне проекта
   - Добавьте:
     ```
     DEEPSEEK_API_KEY=ваш_ключ_здесь
     OPENALEX_EMAIL=ваш_email@example.com
     ```

3. **Запустите приложение:**
   ```bash
   streamlit run app.py
   ```

## Рабочий процесс

### Создание курса
1. Перейдите на страницу "Управление курсами"
2. Создайте новый курс
3. Заполните контекст курса (важно для согласованности лекций)

### Создание лекции
1. Перейдите на "Мастер лекций"
2. Выберите курс
3. Заполните метаданные (название, ключевые слова)
4. **Загрузите источники** (PDF/DOCX/TXT) - это приоритет!
5. Сгенерируйте библиографию через OpenAlex
6. Создайте резюме библиографии
7. Сгенерируйте план
8. Создайте черновик → финальная версия
9. Извлеките глоссарий
10. Сгенерируйте промпт для Gamma

## Важные замечания

- **Загруженные PDF имеют наивысший приоритет** - система будет опираться на них в первую очередь
- Все промежуточные результаты сохраняются в `outputs/{course_id}/`
- Файлы загружаются в `data/uploads/{course_id}/{lecture_id}/`

## Структура выходных файлов

Для каждой лекции создаются:
- `{lecture_id}_sources.json` - резюме загруженных источников
- `{lecture_id}_bibliography.json` - библиография OpenAlex
- `{lecture_id}_bibliography_summary.md` - резюме библиографии
- `{lecture_id}_outline.md` - план лекции
- `{lecture_id}_draft.md` - черновик
- `{lecture_id}_final.md` - финальная версия
- `{lecture_id}_glossary.md` - глоссарий
- `{lecture_id}_gamma_prompt.md` - промпт для Gamma
