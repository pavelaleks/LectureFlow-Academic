Этот файл содержит инструкции по созданию контекста курса.

Контекст курса должен включать:

1. Общую тематику и цели курса
2. Методологические подходы
3. Ключевые концепции и термины
4. Связи с другими курсами или дисциплинами
5. Ожидаемый уровень подготовки студентов

Контекст используется при генерации всех лекций курса для обеспечения согласованности и преемственности.
